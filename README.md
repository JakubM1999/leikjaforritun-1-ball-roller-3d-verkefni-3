# Godot 3D Ball Roller Course Slides

Step-by-step tutorial slides for making a 3D Ball Roller game in Godot.

## Theme
Early 2000s 3D (Y2K / Frutiger Aero precursor) aesthetic.
- Glossy gradients
- 3D floating shapes (Three.js)
- Rounded "pill" buttons

## Structure
- `index.html`: Main Menu
- `intro.html`: Introduction to 3D concepts
- `guide.html`: Step-by-step tutorial
- `style.css`: Theme styling
- `script.js`: Slide navigation logic
- `bg.js`: Three.js background animation

## How to use
Open `index.html` in any modern web browser.
Use Arrow keys or on-screen buttons to navigate slides.
