BALL ROLLER 3D

FULLUR LEIÐARVÍSIR

Skref fyrir skref leiðbeiningar frá byrjun til enda.

1. Nýtt Verkefni

Opnaðu Godot og búðu til nýtt verkefni.

  • Veldu Forward+ eða Mobile renderer.
  • Gefðu verkefninu nafn (t.d. "Ball Roller 3D").
  • Smelltu á Create & Edit.

2. Sigling um 3D heiminn

Áður en við byrjum, lærum að hreyfa okkur í 3D glugganum:

  • Hægri músarhnappur: Snúa myndavél (Orbit).
  • Miðju músarhnappur (eða Shift + Hægri): Hliðra (Pan).
  • Skrunhjól: Aðdráttur (Zoom).
  • F-takki: Fókusa á valda nóðu.

3. Aðalsenan (Main)

Búðu til grunninn að leiknum.

  • Smelltu á 3D Scene í Scene flipanum.
  • Endurskírðu nóðuna í Main.
  • Vistaðu senuna sem main.tscn.

4. Gólfið (Mesh)

Við þurfum palla fyrir boltann.

  1. Bættu við MeshInstance3D sem barn af Main.
  2. Í Mesh eiginleikanum hægra megin, veldu New BoxMesh.
  3. Stökktu Scale í: 20, 1, 20.

5. Gólfið (Eðlisfræði)

Mesh eitt og sér er bara draugur, boltinn myndi falla í gegnum það.

  1. Veldu MeshInstance3D nóðuna.
  2. Efst í glugganum, smelltu á Mesh -> Create Triamesh Static Sibling.
  3. Þetta býr til StaticBody3D með CollisionShape3D.

6. Sólin (Lýsing)

Án ljóss er allt svart.

  • Bættu við DirectionalLight3D.
  • Færðu það upp og snúðu því þannig að það lýsi á gólfið.
  • Kveiktu á Shadows í Inspector til að fá skugga.

7. Umhverfið (Environment)

Gefum leiknum himin og fallega liti.

  • Bættu við WorldEnvironment.
  • Í Environment veldu New Environment.
  • Stillingar: Background Mode -> Sky.
  • Sky -> New Sky -> New ProceduralSkyMaterial.

8. Leikmaðurinn (Sena)

Búðu til nýja senu (Ctrl + N) fyrir boltann.

  • Root nóða: CharacterBody3D.
  • Skírðu nóðuna Player.
  • Vistaðu sem player.tscn.

9. Boltinn (Útlit)

Gerum boltann sýnilegan.

  • Bættu MeshInstance3D við Player.
  • Veldu New SphereMesh.
  • Prófaðu að bæta við Material undir Surface Material Override til að lita boltann!

10. Boltinn (Árekstur)

Boltinn þarf að vita hvenær hann snertir gólfið.

  • Bættu við CollisionShape3D.
  • Veldu New SphereShape3D.
  • Gættu þess að stærðin passi við Mesh-ið.

11. Stýringar (Input Map)

Farðu í Project -> Project Settings -> Input Map.

Búðu til eftirfarandi aðgerðir og tengdu þær við lykla:

  • move_forward (t.d. W eða Up)
  • move_back (t.d. S eða Down)
  • move_left (t.d. A eða Left)
  • move_right (t.d. D eða Right)
  • jump (t.d. Space)

12. Forritun (Script)

Bættu Script við Player nóðuna.

extends CharacterBody3D

const SPEED = 7.0
const JUMP_VELOCITY = 5.0

func _physics_process(delta):
    # Gravity
    if not is_on_floor():
        velocity += get_gravity() * delta

13. Forritun (Hreyfing)

Bættu þessu við _physics_process:

    # Jump
    if Input.is_action_just_pressed("jump") and is_on_floor():
        velocity.y = JUMP_VELOCITY

    # Input Direction
    var input_dir = Input.get_vector("move_left", "move_right", "move_forward", "move_back")
    var direction = (transform.basis * Vector3(input_dir.x, 0, input_dir.y)).normalized()
    
    if direction:
        velocity.x = direction.x * SPEED
        velocity.z = direction.z * SPEED
    else:
        velocity.x = move_toward(velocity.x, 0, SPEED)
        velocity.z = move_toward(velocity.z, 0, SPEED)

    move_and_slide()

14. Setja í Main

Farðu aftur í main.tscn.

  • Dragðu player.tscn úr FileSystem yfir í Scene tréð undir Main.
  • Færðu boltann aðeins upp (Y+) svo hann byrji fyrir ofan gólfið.

15. Myndavélin (Camera)

Við þurfum að sjá hvað er að gerast.

  • Bættu Camera3D við Main senuna (EKKI sem barn af Player).
  • Færðu hana í: 0, 5, 10.
  • Hallaðu henni niður (Rotation X: -25).

16. Myndavélar-fylgd

Ef við setjum myndavélina sem barn af boltanum mun hún rúlla með honum (mjög ruglandi!).

Við notum script til að hún fylgi boltanum mjúklega.

  • Búðu til script á Camera3D.
  • Búðu til breytu: @export var target: Node3D

17. Camera Script

extends Camera3D

@export var target: Node3D
@export var offset = Vector3(0, 5, 10)

func _process(delta):
    if target:
        var target_pos = target.global_position + offset
        global_position = global_position.lerp(target_pos, delta * 5.0)
        look_at(target.global_position)

Mundu að velja Player í Target eiginleikanum í Inspector!

18. Peningar (Coins)

Búðu til nýja senu fyrir pening.

  • Root nóða: Area3D (Area3D er notað til að skynja snertingu).
  • Bættu við MeshInstance3D (CylinderMesh) og CollisionShape3D (CylinderShape3D).
  • Skírðu nóðuna Coin.

19. Snúningur á pening

Gerum peninginn lifandi með scripti.

extends Area3D

func _process(delta):
    rotate_y(delta * 3.0)

func _on_body_entered(body):
    if body.name == "Player":
        # Hér gætum við bætt við stigum!
        queue_free() # Peningurinn hverfur

Tengdu body_entered signalið í Node flipanum!

20. Hönnun borðsins

Nú er kominn tími til að breyta þessu í alvöru leik!

Notaðu verkfærin til að búa til skemmtilega braut:

  • Ctrl + D: Fjölfalda palla (Duplicate).
  • E: Snúa pöllum (Rotate) til að búa til brekkur.
  • R: Stækka/minnka palla (Scale).

Verkefni: Búðu til braut sem er allavega 5 pallar að lengd og með einni beygju.

BÓNUS 1: Hreyfanlegir Pallar

Gerum pall sem fer upp og niður.

  1. Breyttu StaticBody3D í AnimatableBody3D (hægri smelltu -> Change Type).
  2. Bættu við AnimationPlayer nóðu.
  3. Búðu til nýja hreyfimynd (Animation -> New).
  4. Veldu pallinn, farðu í Transform -> Position í Inspector og ýttu á lykilinn .
  5. Færðu tímalínuna, færðu pallinn og ýttu aftur á lykilinn.
  6. Kveiktu á Autoplay (A+ takkinn í Animation glugganum).

BÓNUS 2: Markmarkið

Hvernig vinnur maður leikinn?

  1. Búðu til Area3D sem heitir Goal með áberandi Mesh.
  2. Tengdu body_entered signalið við script.
func _on_goal_body_entered(body):
    if body.name == "Player":
        print("ÞÚ VANNST!")
        # Eða endurræsa leikinn:
        get_tree().reload_current_scene()
